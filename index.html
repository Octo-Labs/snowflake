How do I get started with Snowflake?

It couldn't be easier.

Remove other frameworks

If you're starting a new project, just make sure not to bring in any new frameworks.

Start using Snowflake

There's no library to add to start using Snowflake. Just make up your app as you go.

Yes, it's just that easy!

In fact, unless you've prevented it, you may be using Snowflake already! Isn't that exciting?

Just go with your gut! Start Snowflaking today!
